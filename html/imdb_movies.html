IMDB_Movies Table


The IMDB_Movies table contains the movie data uploaded from the IMDB Movies Extensive Dataset at Kaggle.com. The source *.csv file contains 85,855 records and 22 columns.

Table Structure

Column Name Column Type Description
unique_id bigint Unique identifier
movie_title text Computed column concatenating the year
and title, used to calculate the numeric unique_id via
the hash function.
imdb_title_id text title ID on IMDb
title text title name
original_title text original title name
year bigint Year of release
genre text Movie genre
duration bigint Duration in minutes
country text Movie country
