Series Table


The Series table contains the tv shows data uploaded from the TV shows on Netflix, Prime Video, Hulu and Disney+ at Kaggle.com. The source *.csv file contains 5,564 records and 11 columns.

Table Structure

Column Name Column Type Description
unique_id integer Unique identifier
title text The Movie Title
year integer The year of release
age integer Target Age Group
imdb numeric IMDB index
rotten_tomatoes text Rotten tomatoes rating %
netflix integer If the movie is in Netflix: 0 = false, 1 = true
hulu integer If the movie is in Hulu: 0 = false, 1 = true
prime_video integer If the movie is in Amazon Prime Video: 0 = false, 1 = true
disney_plus integer If the movie is in Disney+: 0 = false, 1 = true
movie_title Text

Computed column, concatenating the title and year.

Used to produce the primary key by hash() function.
