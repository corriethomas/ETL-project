ETL Project

In this project we combined datasets from Kaggle that show film titles, which streaming service(s) the film is on, ratings, imdb scores, and other information specific to the film (director, release year, actors, etc).

We combined two datasets into one relational table in a database to run specific queries. You could use this database to run queries like which streaming service has any particular movie, which service provides more kid friendly movies, etc.

The team used Jupyter Notebook to extract, clean and structure the data. We created a unique id (using the hash function) for the movie titles to be able to join the two datasets into a single table and run these queries.

We used PostgreSQL to create our database. The description of the tables in this database are also published in this website, along with our final report.
